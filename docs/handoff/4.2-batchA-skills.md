4. **倾国 (qingguo)** — Zhen Ji

   - Locked: true (mandatory query)

6. **马术 (mashu)** — Ma Chao
   - Type: `distanceModifier`
   - Distance to Ma Chao is reduced by 1 (others are 1 seat closer)
   - Locked: true (mandatory query)

7. **英姿 (yingzi)** — Zhou Yu
   - Type: `drawCount` → current + 1
   - Draws 1 extra card in draw phase
   - Locked: true (mandatory query)

8. **奇才 (qicai)** — Huang Yueying
   - Type: `ignoresDistance`
   - Trick cards ignore distance limits
   - Locked: true (mandatory query)

9. **谦逊 (qianxun)** — Lu Xun
   - Type: `targetable` (AND-folded)
   - Cannot be targeted by dismantle or indulgence
   - Locked: true (mandatory query)

10. **奇袭 (qixi)** — Gan Ning
    - Type: `cardsAs(dismantle)`
    - Black cards can be played as dismantle
    - Locked: false

### Trigger Skills (2)

11. **闭月 (biyue)** — Diao Chan
    - Event: `phase.start(end)` → draw 1
    - Optional trigger that fires at the start of the end phase
    - Draws an extra card

12. **克己 (keji)** — Lu Meng
    - Event: `phase.end(action)` → skip discard
    - Optional trigger that fires at the end of the action phase
    - **Important:** Checks `G.turnFlags.strikeUsedInAction` (NOT `strikesPlayed`)
    - Per skill-trigger-design.md §2.2, this flag is set whenever a strike is played OR supplied while in the action phase
    - Skips the discard phase if no strike was used/played

## Design Notes

### 克己 Counter Implementation

Per skill-trigger-design.md §2.2:

> "克己's counter is NOT `strikesPlayed`. The rule is '若你未于出牌阶段**使用或打出**过杀' — a 杀 *played in response* (to a 决斗, or via 激将) counts too."

The implementation checks `G.turnFlags.strikeUsedInAction`, which is maintained by the engine whenever a strike is played or supplied while `G.turnPhase === 'action'`. This is distinct from `G.turnFlags.strikesPlayed` which only counts cards played through `playCard`.

### Query Fold Validation

All locked-only queries (strikeLimit, drawCount, demandCount, targetLimit) are correctly declared as `locked: true`, per queryTypes.ts's `assertQueryProvider` check that runs at server boot.
